## v0.1.x {#v0.1}

实现以下基础服务：

- 服务网关：Api 和 Web 资源网关
- 配置中心：
- 认证与鉴权：资源权限与数据权限
- 前后端本地化机制
- ~~DSL 设计器：含设计和管理~~
- 以填写表单形式管理和配置应用结构

设计与实现要点：

- 抽象与实现分离：以**最小化信息表达**实现
  [框架中立性](/blog/the-framework-agnostic-in-reversible-computation)

- 控制外部请求的**守门人**
- 对用户进行认证，对操作进行鉴权，通过状态机驱动资源访问
- 动态生成用户的 Web 端页面，并通过 Nop Xpl 生成 `index.html`
  以便于注入真实的`平台名称`、`平台 Logo`、`等待动画`等定制内容
  - 通过 Nop 的 `precompile` 机制，调用 `npm` 对页面渲染引擎进行构建，
    并将构建产物复制到 `src/main/resources/META-INF/resources`
    目录下
